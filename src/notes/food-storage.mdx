# Peppers

- Ziploc bag in crisper drawer.
	- Can last up to 3 weeks.
- Freezer
- Sun dehydration
	- Wash, Cut into pieces, use a needle and thread to create a strand of peppers and hang them in a sunny area to dry.
	- Store in freezer or airtight container
	- Re-hydrate with warm water before using them.
